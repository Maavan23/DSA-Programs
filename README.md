# Data Structures & Algorithms (DSA) – Java

This repository contains my **Data Structures and Algorithms (DSA) programs** developed as part of my **Software Engineering studies**.

The repository is organized **topic-wise** to make learning, revision, and navigation easy.  
Each folder focuses on a specific DSA concept and includes Java programs, diagrams, and supporting materials.

---

## 📚 Topics Covered

- **Basic Programming**
  - Variables, input/output, arithmetic operations
- **Decision Control Statements**
  - Conditional logic (if, if-else, switch)
- **Loops**
  - Iterative logic and repetition
- **Arrays**
  - Array operations and analysis
- **Searching Algorithms**
  - Linear Search, Binary Search
- **Sorting Algorithms**
  - Bubble Sort, Insertion Sort, Merge Sort
- **Recursion & Dynamic Programming**
  - Fibonacci (recursive, iterative, DP)
- **Linked Lists**
  - Singly linked list implementations
- **Stacks**
  - Stack operations using arrays / lists
- **Queues**
  - Queue implementations
- **Trees**
  - Tree structures and traversal basics
- **Graphs**
  - Adjacency list and adjacency matrix representations
- **Strings**
  - String manipulation problems
- **Flowcharts & Diagrams**
  - Visual representations of algorithms
- **Excel Data & Algorithm Visualizations**
  - Spreadsheet-based representations (e.g., Dijkstra’s Algorithm calculations)
